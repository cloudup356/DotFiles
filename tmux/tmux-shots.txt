TMUX:
------------------------
leader button = CTRL + a


------------------------
start tmux = tmux

make a new window = Alt + c

move to next window = Alt + n

close window = Alt + x

previous window = Alt + p

close a pane = leader + x

move to a different window = leader + (number of window)

split vertical = leader + %

split horizontal = leader + "

move pain = leader + UP, DOWN, LEFT, RIGHT

command mode = leader + :

change window name = (command mode) rename-window (new window name)

detach tmux session = leader + d

list tmux sessions = tmux ls

attach to tmux session = tmux attach

to make a new tmux session = (exit tmux) tmux

list all sessions within tmux = leader + s

to rename session = (command mode) rename-session (new session name)
